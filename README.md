# CI/CD Pipeline Project 

##  Project Overview
This project demonstrates a simple CI/CD pipeline using GitHub Actions.

##  Tools Used
- GitHub
- GitHub Actions
- GitHub Pages

##  CI/CD Workflow
1. Code pushed to GitHub
2. Pipeline triggered automatically
3. Files validated
4. Deployment to GitHub Pages
